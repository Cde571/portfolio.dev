# Featured Projects + Archivo Compacto

Este parche organiza el portafolio para que la pagina principal muestre solo los proyectos mas fuertes y no se sature con las 54 tarjetas.

## Cambios

- 10 proyectos destacados visibles en la seccion principal.
- Boton directo a todos los repositorios de GitHub.
- Archivo local compacto con los 54 repositorios.
- Buscador y filtros por lenguaje dentro del archivo compacto.
- Miniaturas conservadas para cada repo.
- La UI arcade/pixel se mantiene.
